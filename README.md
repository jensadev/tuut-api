# tuut api

Baserat på https://www.youtube.com/watch?v=WfCJ3sHnLBM

## Starta

    git clone
    cd tuut-api
    npm install
    cp .env_example .env
    nano .env

    mysql -u -p
    create database tuut-api;
    exit
    mysql -u -p tuut.-api < tuut.sql

    nodemon start
